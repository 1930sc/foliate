Foliate

A simple and modern eBook viewer

Source Code · Issues

Features

View eBooks in the EPUB format. Foliate is based on Epub.js, and supports both two-page view and scrolled view. Customize font and line-spacing. Choose between light, sepia, dark, and invert mode.

Navigation is easy. View the table of contents, or use the find in book feature. Reading progress slider with chapter marks makes it easy to find your way through the book.

Quickly lookup definitions. Foliate uses metadata included in the eBook to determine which language to lookup. This feature is powered by Wiktionary, the free dictionary.

Add bookmarks, highlight, and notes. Folio stores your reading progress, bookmarks, and annotations in your XDG data directory as plain JSON files, so you can export or sync them easily.

Installation

Manual Installation

Dependencies: gjs, webkit2gtk, libsoup, meson 

git clone https://github.com/johnfactotum/foliate.git
cd foliate
meson build --prefix=/usr
cd build
ninja
sudo ninja install
